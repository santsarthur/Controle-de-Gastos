# Requisitos do Sistema

## Requisitos Funcionais

- RF01: O sistema deve permitir o cadastro de usuários
- RF02: O sistema deve permitir login de usuários
- RF03: O usuário deve cadastrar gastos
- RF04: O usuário deve visualizar a lista de gastos
- RF05: O usuário deve editar gastos
- RF06: O usuário deve excluir gastos
- RF07: O sistema deve calcular o total de gastos
- RF08: O usuário deve categorizar os gastos

## Requisitos Não Funcionais

- RNF01: O sistema deve ser acessível via navegador web
- RNF02: O sistema deve possuir interface simples e intuitiva
- RNF03: O sistema deve responder em até 2 segundos
- RNF04: Os dados devem ser armazenados de forma persistente
- RNF05: O sistema deve garantir autenticação básica
