# Controle de Gastos Pessoais

Sistema web para auxiliar no controle financeiro pessoal.

## ODS
ODS 8 - Trabalho Decente e Crescimento Econômico

## Problema
Muitas pessoas possuem dificuldades em controlar seus gastos mensais, o que pode levar ao endividamento.

## Objetivo
Permitir que usuários registrem, organizem e acompanhem seus gastos de forma simples.

## Tipo de Solução
Aplicação web com frontend e backend.

## Tecnologias
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Banco de Dados: PostgreSQL
